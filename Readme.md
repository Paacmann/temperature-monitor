Temperature and humidity monitoring application for STM32F103.

## Features

- HTU21D temperature and humidity sensor
- SSD1306 OLED display
- Custom portable display driver (Git submodule)
- STM32 HAL integration
- I2C communication

## Hardware

- STM32F103C8T6 (Blue Pill)
- HTU21D
- SSD1306 OLED (128x64)
